Binary Tree
Each node can have most 2 children

 Strict / Proper
 Each node can have either 2 or 0 children
               [ Root ]
                /    \
            [Node]  [Node]
                    /    \
                [Leaf] [Leaf]

 Complete Binary Tree
 All levels except possibly the last are completely
 filled and all nodes are as left as possible
                 [ Root ]            
                  /    \
              [Node]  [Node]          
              /    \    /
          [Leaf][Leaf][Leaf]

 Perfect Binary Tree
 Maximum no. of nodes in a tree with heigh
                 [ Root ]             
                  /    \
              [Node]  [Node]         
              /    \  /    \
          [Leaf][Leaf][Leaf][Leaf]

We can implement binary tree using
- dynamically created nodes
- arrays (used for heaps)

    Binary Search Tree
Array 
 - Search(x) O(n)
 - Insert(x) O(1)
 - Remove(x) O(n)

Linked List
- Search(x) O(n)
- Insert(x) O(1)
- Remove(x) O(1)

BST
- Search(x) O(log n)
- Insert(x) O(log n)
- Remove(x) O(log n)

Binary Search Tree (BST)
A binary tree in which for each node, value of
all the nodes in left subtree is lesser or equal
and value of all nodes in right subtree is greater
Eg:
               [ 15 ]
              /      \
          [ 10 ]    [ 20 ]
          /    \          \
       [ 8 ]  [ 12 ]     [ 25 ]
